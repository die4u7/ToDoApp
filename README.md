# TodoApp

这是一个基于 Android 的 Todo List 应用，用于实现基础待办事项管理功能。

## 功能简介
- 添加待办事项（标题必填，描述可选）
- 删除待办事项
- 标记任务完成 / 未完成
- 任务分类（Work / Study / Life）
- 搜索待办事项
- 本地数据持久化（Room + SQLite）

## 技术栈
- Kotlin
- Jetpack Compose
- MVVM 架构
- Room 数据库
- Kotlin Coroutines & Flow

## 运行方式

1. 使用 **Android Studio** 打开本项目根目录  
2. 等待 Gradle 自动下载依赖并完成构建  
3. 启动 Android 模拟器或连接真实 Android 设备  
4. 点击 Android Studio 中的 **Run** 按钮运行应用  

应用将自动安装并运行在选定的设备上。

## 运行环境
- 操作系统：macOS  
- 开发工具：Android Studio  
- Android 版本：Android 13 / Android 14（模拟器测试）
